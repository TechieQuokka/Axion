# Axion

C# .NET 프로젝트입니다.

## 프로젝트 구조

```
├── src/        # 소스 코드
├── tests/      # 테스트 코드
└── Axion.sln   # 솔루션 파일
```

## 요구사항

- .NET 6.0+
- Visual Studio 2022

## 빌드 및 실행

1. `Axion.sln` 파일을 Visual Studio에서 엽니다
2. 솔루션을 빌드합니다 (Ctrl+Shift+B)
3. 실행합니다 (F5)

## 테스트

```bash
dotnet test
```
